# EC2 Web Server Deployment

## Project Overview
This project demonstrates how to deploy a web server using AWS EC2.

## Steps Performed
1. Created an EC2 instance (Amazon Linux)
2. Connected to instance using SSH
3. Installed Apache web server
4. Started and enabled HTTP service
5. Configured security group to allow HTTP (port 80)
6. Accessed website using public IPv4

## Output
The web server is successfully running and accessible via browser.
